# timberborn-calculator
A simple Timberborn food and production calculator.

[![codecov](https://codecov.io/gh/JudeBake/timberborn-calculator/graph/badge.svg?token=4HADYAN62U)](https://codecov.io/gh/JudeBake/timberborn-calculator)

## Setup
1. Clone the repository
```bash
git clone git@github.com:JudeBake/timberborn-calculator.git
```
2. Create the virtual environment
```bash
python3 -m venv .venv
```
3. Activate the virtual environment
```bash
source .venv/bin/activate
```
